# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

### API Actions

- [x] JWT Authentication
- [x] Login (JWT)
- [x] Create Account
- [x] See Profile
- [x] Edit Profile
- [x] Add Room to Favourites
- [x] List Rooms
- [x] See Room
- [x] Create Room
- [x] Edit Room
- [x] Delete Room
- [x] Search Rooms

### Graphql API Actions

##### Rooms

- [x] List Rooms (with Pagination)
- [] See Room
- [] Add Room to Favourites

#### Users

- [] Login (JWT)
- [] Create Account
- [] See Favs
- [] See Profile
- [] Edit Profile
